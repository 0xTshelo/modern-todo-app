# Modern To-Do App

A clean and modern to-do list app built with Python and CustomTkinter.

## Features
- Add tasks
- Mark tasks as done
- Delete tasks
- Persistent storage using JSON
- Modern dark UI

## Tech Stack
- Python
- CustomTkinter
